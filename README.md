# @salts/cli

[![npm version](https://img.shields.io/npm/v/@salts/cli?style=flat-square)](https://www.npmjs.com/package/@salts/cli)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg?style=flat-square)](https://opensource.org/licenses/MIT)

A lightweight and efficient CLI utility designed for rapid project scaffolding and command template generation.

## 📌 Features

- **Automated Scaffolding**: Generate boilerplate command structures instantly.
- **Minimalist Design**: Zero-config setup with a focused feature set.
- **Type-Safe**: Built with TypeScript for better developer experience and reliability.

## 🚀 Installation

Install the package globally using NPM:

```bash
npm install -g @salts/cli
```

## 🛠 Usage
Once installed, the `salts` command will be available globally in your terminal.

### Generate Command Template
Create a new command boilerplate by providing a name:

```sh
salts g <command-name>
```

## 📂 Local Development
If you wish to contribute or test the CLI locally, follow these steps:

1. **Clone this repository:**
   ```sh
   git clone https://github.com/VonAntaraxia/salts-cli
   ```

3. **Install dependencies:**
   ```sh
   npm install
   ```

4. **Build the project:**
   ```sh
   npm run build
   ```

5. **Link the package locally:**
   ```sh
   npm link
   ```
